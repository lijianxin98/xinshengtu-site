服务流程

从体检报告到志愿落地，再到大学四年规划
完整陪伴，全程可见，让每一步都有保障

四大交付物：完整服务流程

服务时间轴与进度管理

D1

体检报告（问卷驱动）

先做家庭与学生的全面画像，再定制志愿方案

飞书问卷采集三大维度：家庭约束、能力画像、路径倾向

家庭约束

城市、成本、距离、类型

能力画像

能力、风格、性格分析

路径倾向

考研/就业/出国/考公

冲突诊断

目标不一致、预算冲突提示

交付物清单：

  • 家庭约束清单报告
  • 学生画像雷达图
  • 路径倾向分析报告
  • 冲突诊断与沟通建议
D2

专业性展示报告（基准版）

单方面生成，先立专业感

基于分数位次 + 体检报告，输出可行区间 + 方向建议 + 风险清单

可行区间

冲/稳/保区间明确边界

风险提示

政策、体检、单科限制

取舍框架

收益与代价说明

方向建议

初步院校/专业参考

交付物清单：

  • 可行区间分析报告
  • 风险清单（需二次核对项）
  • 初版方向建议书
  • 取舍解释说明文档
D3

志愿落地方案（共创权重版）

核心成交交付，冲/稳/保三套方案

家长选择侧重维度，AI多目标权衡，输出冲/稳/保三套志愿表

候选池

筛选符合条件院校专业

三套志愿表

冲/稳/保 + 理由说明

风险清单

标注潜在风险与应对

核对清单

关键项目二次确认

交付物清单：

  • 冲档志愿表（含理由与风险）
  • 稳档志愿表（含理由与风险）
  • 保档志愿表（含理由与风险）
  • 综合风险清单
  • 提交前二次核对清单
D4

大学四年规划（陪跑）

长期高端护城河，可见成果交付

以可见成果为核心：论文、项目、证书、实践、作品集、简历等

路径目标

主路径+备选设定

关键动作

课程/证书/项目/实习节奏

成果清单

作品、项目、实习、证书

督管复盘

1v1督管 + 决策支持 + 复盘

交付物清单：

  • 四年成长路线图
  • 每学期关键动作清单
  • 可见成果模板与指导
  • 一对一督管与复盘服务
  • 年度成长评估报告

服务保障

证据可追溯

每个推荐都有依据说明，规则、政策来源可查，让决策过程透明可信

人工复核

低置信项、敏感规则进入人工复核队列，确保严谨性与可靠性

持续迭代

标注回流形成经验库，不断优化规则与方法论，提升服务质量

全程沟通

体检报告后沟通确认，权重共创面对面交流，重要决策点及时反馈

典型服务时间轴

从高考出分到志愿填报，再到大学四年，全程陪伴

T0

出分前准备

飞书问卷采集，生成D1体检报告，家庭沟通确认

T+1

出分后24小时

生成D2专业性展示报告，明确可行区间与风险点

T+2

权重共创

家长选择主维度，面对面交流取舍逻辑

T+3

方案交付

D3志愿落地方案（冲/稳/保三套），风险清单与核对清单

T+5

提交前复核

二次核对清单逐项确认，确保无遗漏

T+30

录取后规划

D4大学四年规划启动，路线图与成果清单交付

4年

全程陪跑

每学期关键动作督管，年度复盘，可见成果积累
