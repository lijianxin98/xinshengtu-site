## 功能特色

### 核心价值
1. **方向适配** - 分数+人+家庭+未来路径综合最优
2. **成本可控** - 城市/学费/距离/资源全维度评估
3. **路径清晰** - 就业/考研/出国/考公路径量化
4. **风险管控** - 政策细则/体检限制/科目要求全面核查

### AI辅助决策
- **信息处理提效**: 处理速度提升12倍（24h → 2h）
- **多目标权衡**: 6维度智能取舍（录取/适配/成本/升学/就业/风险）
- **可解释输出**: 家长语言解释每个志愿排序逻辑
- **复核闭环**: 低置信度进入人工复核队列

### 四大交付物
- **D1 体检报告**: 家庭约束清单 + 学生能力雷达 + 路径倾向诊断
- **D2 专业展示报告**: 可行区间 + 风险清单 + 初版方向建议
- **D3 志愿落地方案**: 冲/稳/保三套方案 + 风险清单 + 核对清单
- **D4 大学四年规划**: 路径目标 + 每学期关键动作 + 可见成果清单

## 部署指南

### 方式一：Vercel CLI（推荐）
```bash
# 安装Vercel CLI
npm install -g vercel

# 登录
vercel login

# 部署到生产环境
vercel --prod
```

### 方式二：GitHub自动部署
```bash
git add .
git commit -m "v2.5: 图表配色丰富化"
git push origin main
```

然后在Vercel Dashboard中连接GitHub仓库，自动部署。

### 方式三：一键脚本
```bash
chmod +x deploy-original.sh
./deploy-original.sh
```

## 自定义域名配置

### DNS设置（阿里云示例）
1. 登录阿里云域名控制台
2. 添加CNAME记录：
   - 主机记录: `www`
   - 记录类型: `CNAME`
   - 记录值: `cname.vercel-dns.com`
   - TTL: `10分钟`

### Vercel配置
1. 进入项目设置 → Domains
2. 添加域名: `www.xinshengtu.com`
3. 按提示完成DNS验证
4. 等待5-10分钟生效
