## 🎭 设计细节

### 卡片结构
```html
<div class="capability-card">
    <!-- 右上角编号 -->
    <div class="capability-number">01</div>
    
    <!-- 大图标（80×80） -->
    <div class="capability-icon">
        <i class="fas fa-clipboard-list"></i>
    </div>
    
    <!-- 标题（22px） -->
    <h3>飞书问卷引擎</h3>
    
    <!-- 描述段落 -->
    <p>模块化题库设计...</p>
    
    <!-- 特性列表（4项） -->
    <ul class="capability-features">
        <li>模块化题库</li>
        <li>自动计分</li>
        <li>冲突诊断</li>
        <li>一键生成</li>
    </ul>
</div>
```

### 动画时序
```
悬停触发：
├─ 0ms: 顶部渐变条展开（0.4s）
├─ 0ms: 卡片上浮12px（0.4s cubic-bezier）
├─ 0ms: 阴影加深（0.4s）
├─ 0ms: 边框淡入（0.4s）
└─ 0ms: 图标放大旋转（0.4s）
     └─ 启动脉冲动画（2s infinite）
```

### CSS关键技术
```css
/* 顶部渐变条（悬停展开） */
.capability-card::before {
    transform: scaleX(0);
    transform-origin: left;
}
.capability-card:hover::before {
    transform: scaleX(1);
}

/* 图标光晕脉冲 */
@keyframes pulse {
    0%, 100% { transform: scale(1); opacity: 0.3; }
    50% { transform: scale(1.1); opacity: 0.5; }
}

/* 平滑过渡曲线 */
transition: all 0.4s cubic-bezier(0.4, 0, 0.2, 1);
```

## 📝 六大能力详细内容

### 能力1：飞书问卷引擎
- 图标：`fa-clipboard-list`
- 描述：模块化题库设计，智能计分与冲突诊断，自动生成结构化体检报告
- 特性：
  1. 模块化题库（家庭/能力/路径）
  2. 自动计分与量化输出
  3. 冲突诊断与提示
  4. 体检报告一键生成

### 能力2：分数位次与规则引擎
- 图标：`fa-calculator`
- 描述：精准确定可行集合，智能计算冲/稳/保区间，严格匹配硬性约束条件
- 特性：
  1. 历年分数位次对比
  2. 冲/稳/保智能区间划分
  3. 政策规则自动匹配
  4. 硬性约束条件筛选

### 能力3：志愿目标多维评估
- 图标：`fa-chart-bar`
- 描述：六大维度全面评估，院校专业智能打标签，排序逻辑清晰可解释
- 特性：
  1. 录取/适配/成本/升学/就业/风险
  2. 院校专业智能标签
  3. 多维度排序算法
  4. 推荐理由自动生成

### 能力4：多需求权重共创面板
- 图标：`fa-sliders-h`
- 描述：家长自主选择关切维度，AI智能权衡多目标，生成个性化方案对比
- 特性：
  1. 家长Top3关切维度
  2. 权重滑块可视化调节
  3. 冲/稳/保三套方案对比
  4. 取舍说明清晰透明

### 能力5：交付物生成器
- 图标：`fa-file-alt`
- 描述：统一模板标准化输出，体检报告、志愿方案、核对清单一键生成
- 特性：
  1. 体检报告PDF导出
  2. 志愿表Excel生成
  3. 风险清单结构化
  4. 核对清单自动输出

### 能力6：大学四年成长系统
- 图标：`fa-graduation-cap`
- 描述：路径目标与志愿一致，成果清单、阶段复盘，全程陪跑督管服务
- 特性：
  1. 四年成长路线图
  2. 每学期关键动作清单
  3. 可见成果模板指导
  4. 一对一督管复盘
