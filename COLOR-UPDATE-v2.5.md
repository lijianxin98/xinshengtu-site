## 配色设计原则

### 1. 色彩语义
- **蓝色/青色**: 科技、AI、专业、可信
- **紫色**: 创新、智慧、高端
- **绿色**: 成长、稳定、积极
- **橙色**: 活力、警示、对比
- **红色**: 重要、强调、品牌

### 2. 对比原则
- 传统 vs AI: 使用对比色（灰vs蓝、橙vs蓝）
- 不同分类: 使用邻近色或互补色
- 时间进度: 使用渐变色系

### 3. 视觉层次
- 主要数据: 饱和度高的颜色（蓝、紫、青）
- 次要数据: 中等饱和度（浅蓝、浅绿）
- 背景/对比: 低饱和度（灰色）

### 4. 品牌一致性
- 保留品牌红色作为强调色
- 其他颜色作为功能性配色
- 整体色调和谐统一

## 技术实现

### Chart.js配置
```javascript
// 全局字体设置
Chart.defaults.font.family = 'PingFang SC, Microsoft YaHei, sans-serif';

// 统一配色方案
const colorPalette = {
    blue: '#2E86AB',
    purple: '#6A4C93',
    teal: '#118AB2',
    orange: '#FF6B35',
    green: '#2D6A4F',
    // ... 其他颜色
};

// 应用配色
backgroundColor: [colorPalette.orange, colorPalette.blue]
```

### 响应式设置
- 所有图表启用 `responsive: true`
- 容器固定高度防止布局跳动
- 移动端自动调整图例和标签大小
