交付物
D1

体检报告（问卷驱动）

先做家庭与学生的全面画像，再定制志愿方案

通过飞书问卷采集：家庭约束 + 学生能力 + 路径倾向 → 生成量化体检报告

家庭硬约束清单

  • 城市偏好与范围
  • 成本预算上限
  • 离家距离要求
  • 学校类型接受度（公办/民办/中外合作）

学生能力画像雷达

  • 学科优势与短板
  • 学习风格与习惯
  • 性格偏好（外向/内向/动手/动脑）
  • 不贴标签，只做客观描述

路径倾向与冲突诊断

  • 量化路径倾向（考研/就业/出国/考公）
  • 识别目标不一致情况
  • 预算与路径冲突提示
  • 风险偏好不匹配预警

沟通清单与建议

  • 家长与学生需对齐的关键问题
  • 潜在分歧点提示
  • 沟通话术建议
  • 为后续权重共创做铺垫

D1 交付物清单

家庭约束清单报告
学生画像雷达图
路径倾向分析报告
冲突诊断与沟通建议
交付物
D2

专业性展示报告（基准版）

单方面生成，先立专业感

基于分数位次 + 体检报告 → 可行区间 + 方向建议 + 风险清单 + 取舍解释

可行区间与边界条件

  • 根据分数位次确定"冲/稳/保"区间
  • 明确可报考院校专业范围
  • 边界条件说明（最低保底线）

关键风险提示

  • 政策细则（如单科要求）
  • 体检限制（色弱/身高等）
  • 专业特殊要求

取舍解释框架

  • 院校层次 vs 专业匹配的权衡
  • 城市选择的收益与代价
  • 不同路径的利弊分析

初版方向建议

  • 结合体检报告的初步方向
  • 参考院校与专业组推荐
  • 说明推荐依据与逻辑

可行区间分析报告
风险清单（需二次核对项）
初版方向建议书
取舍解释说明文档
交付物
D3

志愿落地方案（共创权重版）

核心成交交付，冲/稳/保三套志愿表

家长选择侧重维度 → AI多目标权衡 → 冲/稳/保三套方案 + 核对清单

候选池与筛选依据

  • 从海量院校专业中筛选符合条件的候选集
  • 筛选依据：分数匹配 + 家庭约束 + 能力适配
  • 保证候选池质量与数量平衡

冲/稳/保三套志愿表

  • 每套方案包含完整志愿排序
  • 逐条说明推荐理由与风险提示
  • 标注院校层次、专业特点、就业方向

综合风险清单

  • 政策风险（单科/体检/特殊要求）
  • 录取风险（大小年波动/冷门专业）
  • 适配风险（能力不匹配/路径冲突）

提交前核对清单

  • 关键信息二次确认项
  • 易漏项提醒（专业代码/批次/调剂）
  • 最后关头避免失误

D4

大学四年规划（持续陪跑）

志愿只是起点，大学四年才是关键

路径目标 + 每学期关键动作 + 可见成果清单 + 一对一督管复盘

路径目标明确

  • 就业方向（行业/岗位/企业类型）
  • 考研路线（保研/考研/专业选择）
  • 出国规划（国家/项目/时间节点）
  • 考公考编（岗位要求/备考策略）

每学期关键动作

  • 必修课程与成绩要求
  • 证书考取（英语/计算机/行业证书）
  • 竞赛参与（学科/创新/创业竞赛）
  • 实习实践（项目/企业/科研）

可见成果清单

  • 论文/作品集/项目成果
  • 实习证明/推荐信
  • 竞赛获奖/荣誉证书
  • 简历材料与面试准备

陪跑督管与复盘

  • 一对一定期沟通（月度/学期）
  • 进度跟踪与调整建议
  • 阶段复盘与经验总结
  • 家庭-学校-机构三方协同
