## 🚀 部署步骤

### 方式一：Vercel CLI（推荐）
```bash
# 1. 确保已安装Vercel CLI
npm install -g vercel

# 2. 登录Vercel
vercel login

# 3. 部署到生产环境
vercel --prod

# 预计耗时：2-3分钟
```

### 方式二：GitHub自动部署
```bash
# 1. 提交更改
git add .
git commit -m "v2.5: 图表配色丰富化 - 橙蓝青紫12色方案"
git push origin main

# 2. Vercel自动构建部署（约3分钟）
```

### 方式三：一键脚本
```bash
chmod +x deploy-original.sh
./deploy-original.sh
```

## 📞 域名配置（如需自定义域名）

### DNS设置
1. 登录域名服务商（如阿里云）
2. 添加CNAME记录:
   - 主机记录: `www`
   - 记录类型: `CNAME`
   - 记录值: `cname.vercel-dns.com`
   - TTL: `10分钟`

### Vercel配置

2. 添加: `www.xinshengtu.com`
