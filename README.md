# 昕升途·志愿智能体 官方网站

> 生涯规划 + 志愿落地 + 大学四年成长陪跑

## 项目概述

这是昕升途（南京）教育信息咨询有限公司的官方网站，采用**高级红色配色体系**，展示公司的志愿填报智能决策系统产品。

网站采用**多页结构 + 首屏信息卡片墙 + 可视化/图标密集设计**，基于产品手册v3.3的AI决策可视化内容，提供专业、现代、易用的用户体验。

## 核心特色

### 🎨 设计特色
- **高级红色配色体系**：主色#C41E3A（主红色）+ #9B1728（深红色）+ #E94B5F（浅红色）
- **信息卡片墙**：首屏采用4张信息卡片展示核心价值
- **图标密集**：全站使用Font Awesome 6.4图标系统，提升视觉表现力
- **响应式设计**：完美适配桌面端、平板和移动端

### 📱 页面结构
1. **首页 (index.html)**
   - 首屏Hero区：一句话定位 + 统计数据卡片
   - 信息卡片墙：4大核心价值（体检驱动、AI辅助、复核闭环、四年陪跑）
   - 核心价值：4个维度展示解决方案
   - 四大交付物：D1-D4交付物概览
   - 家长语言六维：决策维度可视化
   - AI辅助决策：4大核心能力

2. **产品介绍 (product.html)**
   - 三张雷达图：家庭约束、学生画像、路径倾向
   - AI辅助决策四大能力：信息处理、多目标权衡、可解释输出、复核闭环
   - 六大能力模块：问卷引擎、规则引擎、多维评估等
   - 三大特色功能：证据链、权重共创、复核保障
   - 优势对比表：昕升途 vs 常见服务

3. **服务流程 (process.html)**
   - D1 体检报告：问卷驱动，全面画像
   - D2 专业性展示报告：可行区间与风险提示
   - D3 志愿落地方案：冲/稳/保三套方案
   - D4 大学四年规划：陪跑督管与成果清单
   - 服务保障：4项保障承诺
   - 服务时间轴：从出分到大学四年

4. **关于我们 (about.html)**
   - 公司简介：企业背景与定位
   - 使命愿景：核心价值观
   - 服务承诺：5项承诺保障
   - 联系方式：仅保留官网联系方式

## 部署指南

### Vercel部署（推荐）

1. **安装Vercel CLI**
```bash
npm install -g vercel
```

2. **初始化项目**
```bash
vercel
```

3. **配置自定义域名**
- 在Vercel Dashboard添加域名 `www.xinshengtu.com`
- 在阿里云DNS添加以下记录：
  - 类型：CNAME
  - 主机记录：www
  - 记录值：cname.vercel-dns.com

4. **部署更新**
```bash
vercel --prod
```

### 手动部署

1. 将所有文件上传到服务器
2. 确保服务器支持静态文件托管
3. 配置域名指向服务器IP
4. 开启HTTPS（推荐使用Let's Encrypt）
