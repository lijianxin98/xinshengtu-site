## 🔧 修复方案

### 修复 1: 自定义图例标签 ✅

**新代码 (index.html line ~795-815)**:
```javascript
legend: {
    position: 'bottom',
    labels: {
        padding: 12,
        font: { size: isMobile ? 9 : 12 },  // 移动端优化
        usePointStyle: true,
        pointStyle: 'circle',
        // ✅ 添加自定义标签生成器
        generateLabels: function(chart) {
            const data = chart.data;
            if (data.labels.length && data.datasets.length) {
                return data.labels.map((label, i) => {
                    const value = data.datasets[0].data[i];
                    return {
                        text: label + ': ' + value + '%',  // 标签 + 值 + 百分号
                        fillStyle: data.datasets[0].backgroundColor[i],
                        hidden: false,
                        index: i
                    };
                });
            }
            return [];
        }
    }
}
```

**效果**:
- ✅ 图例显示: "直接就业: 35%"
- ✅ 图例显示: "国内考研: 32%"
- ✅ 图例显示: "出国深造: 18%"
- ✅ 图例显示: "考公考编: 12%"
- ✅ 图例显示: "自主创业: 3%"

## 📋 技术说明

### Chart.js generateLabels 函数

`generateLabels` 是 Chart.js 提供的钩子函数，用于自定义图例标签。

**参数**:
- `chart`: Chart.js 实例对象

**返回值**: 数组，每个元素包含：
```javascript
{
    text: '显示的文本',           // 图例文字
    fillStyle: '#FF6B35',         // 颜色
    hidden: false,                // 是否隐藏
    index: 0                      // 索引
}
```

**使用场景**:
- 在图例中显示数据值
- 自定义图例格式
- 添加额外信息（如单位、百分比）
